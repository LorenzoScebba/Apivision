# Directory Structure
```
"C:\xampp\" OR "/var/www/"
│
|---WebFolder (chmod 755)
|   └──Api
|   └──controller
|   └──css
|   └──img (chown www:data:www-data)
|   └──js
|   └──model
|   └──vendor
|   └──view
|
|---cgi-bin
    └──config.ini (chmod 755)
```
